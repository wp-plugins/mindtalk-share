Mindtalk is a social channel. This plugins is Add Mindtalk share button and widget on Wordpress.

== Description ==

This plugin will help you to simplify adding mindtalk button share to your post and and content. You can set where the button showing, just on home page, single post or category page. On single post not only one button at the top, but you can also add button at bottom of content.

== Installation ==

1. Unzip the zipped file and upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Try to adding Mindtalk button on post editor
4. Setting when the button show, go to setting -> Mindtalk button
